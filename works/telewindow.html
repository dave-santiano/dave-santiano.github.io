TeleWindow

2018/Tool/Research

The Project:

TeleWindow is an ongoing telepresence research project being conducted under the guidance of Michael Naimark. The aim is to explore how a two-way, one to one, "telepresence window", can be achieved with pre-existing and emerging technology and computational power. Utilizing these different ingredients, we can prototype a device that can offer features such as two-way eye contact, stereoscopic 3D, and presence.

How it works:

Currently, we are using four Intel RealSense depth cameras in order to create a volumetric pointcloud representation of a person, and then we correct perspective based on the head/eye positions of the user. At the moment, the recreation is only geared towards monoscopic viewing, but future implementations that use lenticular sheets would allow for stereoscopic and headset-less viewing.

My involvement with the project focused on:

1. Implementing a pre-trained neural net based face tracker, and a haar cascade based eye tracker.
2. The off-axis viewing frustum OpenGL shader, so that correct perspectives can be given to a user when viewing the display off-axis
3. Stereo sound routing implemented using the JUCE audio programming framework
