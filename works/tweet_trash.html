Trump Tweet Trasher

2017/Art

The Trump Twitter Trasher is a photograph of Donald Trump that has been modified to spit out his tweets into the trash(which will then be recycled). In a high speed information era of bad news, I offer this as a break for people looking to vent their frustrations on the current state of affairs.

The technology used was a Node.js server running on the back-end, which served up an HTML page and then sent tweet data to an Arduino attached to a receipt printer, where the tweets flowed into a carefully positioned trash can. The basic flow of information was: Twitter —> Node.js server —> HTML page with buttons for selection —> back to the Node.js server —> Arduino —> Receipt Printer —> Trash.
