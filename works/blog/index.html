Web Development Midterm, Happy Halloween!

October 31, 2016

Happy Halloween! So with the semi-completion of this website comes a reflection/devlog of how it came to be, what it needs, interesting roadblocks etc. etc. It's been pretty fun so far, visual projects have always been my favorite since changes and iterations are immediately visible, and the tiniest of details can make huge differences.

What I tried to do with this was make a site inspired by ITP fellow Moon's portfolio website: www.moqn.net I got the basic structure similar but then I saw another website that inspired me in a more visual sense, musician and artist Disasterpeace: disasterpeace.com. I loved the simple visual style to it, and decided to go about imitating it but with my own personal flare.

I animated some icons and a stand-in logo to give the website a bit of dynamic and "quirkiness" so that it was friendly and immediately showcased a tiny fraction of the work that I like to do. It's relatively simple, and honestly the structure of the website is relatively the same amount of complexity as the in-class CSS polaroid thumbnail example. I think I've grasped the concepts of everything pretty well though, and the next steps are what I think will challenge me a bit more and really take the website to the next level.

The future things that I'll be working on is having the landing page image be a photo of a random project that has been finished or one that I am currently working on. I also want to add an additional animation that will provide textual information to the navigation icons on mouse hover. I'll also be adding animated loading icons to take care of the dreadful photo loading screen swipe thing.
